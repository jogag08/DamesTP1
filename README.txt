Game Theory : Checkers


Coopératif ou non-coopératif?
C'est un jeu non-coopératif car il n'y a pas d'ententes possible.


Symétrique ou non?
Le jeu est symétrique car les deux jours ont devant eux le même nombre de pions aux mêmes endroits.


Zero-Sum ou Non Zero-Sum?
C'est un jeu non zero-sum parce que quand un joueur mange un pion, le pion ne fait que disparaître.
Le joueur n'acquiert pas le pion.
